# APT Simulation Framework (Educational Tool)

**Author:** Keerthi  
**Course:** MCA Final Year Project  

## Overview
This framework is a Python-based educational tool designed to simulate the lifecycle of an Advanced Persistent Threat (APT). It follows the common attack phases used by real-world threat actors.

## Attack Phases Simulated:
1. **Reconnaissance**: Information gathering and port scanning.
2. **Initial Access**: Simulating a phishing-based breach.
3. **Command & Control (C2)**: Establishing a communication channel.
4. **Privilege Escalation**: Attempting to gain Administrator rights.
5. **Persistence**: Ensuring the "malware" survives a reboot.
6. **Lateral Movement**: Spreading through the internal network.
7. **Data Collection**: Finding sensitive files.
8. **Data Exfiltration**: Stealing the data.

## How to Run:
1. Open terminal in the project folder.
2. Run `python main.py`.
3. Follow the menu options to simulate the attack.
4. Generate a report to see the full timeline.
